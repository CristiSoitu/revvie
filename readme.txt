
Files with centroids should be in z x y ID format.

revvie has a few functionalities. change the revvie_config.toml file to change the key that triggers those.

e – record a match. Select in vitro and in vivo
d – delete a match. Select in vitro only
s – save state of the system. No selection needed
y – accept predicted match. Select in vitro only
n – reject predicted match. Select in vivo only
r – run automatch. Select any in vitro point on slice
t – go to matched pair quickly. Select in vitro only
